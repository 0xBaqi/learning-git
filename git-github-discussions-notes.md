\# GitHub Discussions Notes



\## What are GitHub Discussions?

GitHub Discussions is a community forum built

into GitHub repositories. It provides a space

for conversations that do not fit into issues

or pull requests.



\## Discussions vs Issues



\### Use Issues For

\- Bug reports

\- Feature requests

\- Specific tasks to complete

\- Things that can be closed when done



\### Use Discussions For

\- General questions and help

\- Ideas and brainstorming

\- Announcements to community

\- Polls and feedback

\- Open ended conversations



\## Discussion Categories



\### Common Default Categories

\- Announcements — project news and updates

\- General — general conversation

\- Ideas — feature brainstorming

\- Q and A — questions with answers

\- Show and Tell — share what you built



\### Custom Categories

\- Create categories for your needs

\- Set emoji and description

\- Choose discussion format

\- Open or closed to new posts



\## Creating a Discussion

1\. Go to repo on GitHub

2\. Click Discussions tab

3\. Click New discussion

4\. Choose category

5\. Add title and content

6\. Submit discussion



\## Q and A Format

\- Special format for questions

\- Answers can be marked as solution

\- Original poster marks correct answer

\- Helps others find solutions quickly

\- Shows resolved status on discussion



\## Polls in Discussions

\- Create polls for community feedback

\- Multiple choice options

\- See vote counts in real time

\- Good for feature prioritization

\- Limited time voting periods



\## Pinning Discussions

\- Pin important discussions to top

\- Good for announcements and FAQs

\- Up to four pinned discussions

\- Manage from discussion menu



\## Converting Issues to Discussions

\- Some conversations belong in discussions

\- Convert issue to discussion easily

\- History and comments preserved

\- Link between issue and discussion kept



\## Enabling Discussions

1\. Go to repo Settings

2\. Scroll to Features section

3\. Check Discussions checkbox

4\. Discussions tab appears in repo



\## Notifications

\- Subscribe to discussions you care about

\- Get notified of new replies

\- Unsubscribe when resolved

\- Manage in GitHub notification settings



\## GitHub Discussions for Open Source

\- Central place for community questions

\- Reduces duplicate issues

\- Builds community around project

\- Welcomes new contributors

\- Documents common problems and solutions



\## Tips

\- Set up categories before announcing

\- Create FAQ discussion for common questions

\- Encourage questions in discussions not issues

\- Respond promptly to build community trust

\- Mark solutions to help future searchers

\- Pin announcements for visibility
