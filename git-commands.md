\# Git Commands Reference



\## Setup

\- git config --global user.name "Name"

\- git config --global user.email "email"

\- git config --list



\## Getting Started

\- git init

\- git clone <url>



\## Staging \& Committing

\- git status

\- git add .

\- git add <file>

\- git commit -m "message"

\- git log

\- git log --oneline



\## Branching

\- git branch

\- git switch -c <branch>

\- git switch <branch>

\- git branch -d <branch>



\## Remote

\- git push origin main

\- git pull

\- git remote add origin <url>



\## Undoing

\- git restore <file>

\- git reset HEAD\~1

\- git revert <hash>
