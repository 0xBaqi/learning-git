\## Lessons Learned



\### Technical Lessons

\- Git is a tool that rewards daily use

\- Small frequent commits beat large rare ones

\- Branching is cheap and should be used often

\- Always pull before starting new work

\- Read error messages carefully before acting



\### Habit Lessons

\- Consistency matters more than perfection

\- Documenting learning reinforces knowledge
