\# Branching in Git



\## What is a branch?

A branch is a separate version of your code.

It lets you work on new features without affecting main.



\## Common commands

\- git branch — list all branches

\- git switch -c feature/name — create and switch to new branch

\- git switch main — go back to main

\- git merge <branch> — merge branch into current

\- git branch -d <branch> — delete a branch



\## Workflow

1\. Create a branch

2\. Make changes

3\. Commit changes

4\. Push to GitHub

5\. Create pull request

6\. Merge into main

7\. Delete branch
