\# GitHub Marketplace Notes



\## What is GitHub Marketplace?

GitHub Marketplace is a store for apps and

actions that extend GitHub functionality.

It has tools for code quality, security,

project management, and CI/CD.



\## Types of Marketplace Items



\### GitHub Apps

\- Integrate third party services

\- Install on repos or organizations

\- Have their own permissions

\- Run on external servers

\- Examples: Slack, Jira, Linear



\### GitHub Actions

\- Reusable workflow steps

\- Use in your CI/CD pipelines

\- Free to use in workflows

\- Thousands available

\- Create your own and publish



\## Finding Apps and Actions



\### Browse Marketplace

\- github.com/marketplace

\- Filter by category

\- Filter by pricing

\- Read reviews and ratings

\- Check verification status



\### Categories Available

\- Code quality and review

\- Security and compliance

\- Testing and coverage

\- Project management

\- Deployment and hosting

\- Monitoring and alerts

\- Dependency management



\## Popular GitHub Apps



\### Code Quality

\- SonarCloud — code quality analysis

\- CodeClimate — maintainability metrics

\- Codecov — test coverage reporting

\- DeepSource — automated code review



\### Project Management

\- ZenHub — agile project management

\- Linear — issue tracking

\- Clubhouse — sprint planning



\### Security

\- Snyk — dependency vulnerability scanning

\- WhiteSource — open source security

\- GitGuardian — secret detection



\### Communication

\- Slack — notifications in Slack

\- Microsoft Teams — Teams integration

\- PagerDuty — incident management



\## Popular GitHub Actions



\### CI/CD

\- actions/checkout — checkout your code

\- actions/setup-node — setup Node.js

\- actions/setup-python — setup Python

\- docker/build-push-action — build Docker images



\### Deployment

\- aws-actions — deploy to AWS

\- azure/webapps-deploy — deploy to Azure

\- google-github-actions — deploy to GCP



\### Code Quality

\- github/super-linter — run multiple linters

\- codecov/codecov-action — upload coverage



\### Notifications

\- slackapi/slack-github-action — Slack alerts

\- peter-evans/create-issue — auto create issues



\## Using Actions in Workflows

\- Search marketplace for action name

\- Copy the uses line from marketplace

\- Add to your workflow yaml file

\- Configure required inputs

\- Action runs in your CI pipeline



\## Publishing Your Own Action

\- Create action.yml metadata file

\- Write action code in JavaScript or Docker

\- Push to public GitHub repository

\- Publish to marketplace from repo settings

\- Others can use your action in their workflows



\## Pricing

\- Many apps and actions are free

\- Some have free tiers with paid upgrades

\- GitHub Actions minutes are free for public repos

\- Private repos get limited free minutes

\- Check pricing before installing



\## Tips

\- Check verification badge for trusted apps

\- Read reviews before installing

\- Review permissions apps request carefully

\- Remove apps you no longer use

\- Prefer official actions from verified publishers

\- Check action source code before using
